# Architecture 阶段输出文件模板

本文件包含 Architecture Skill 产出的文件模板。

**重要约定**：
- 所有过程文档存放在 `{kb_dir}/` 目录下，该目录由调用方指定
- `{name}`：需求/问题名称，用作目录名，英文小写字母，多个单词用 `-` 连接
- 模板中的路径需要替换为实际路径

---
