# 文档输出模板

本文件包含 Doc Skill 使用的文档输出模板。

**重要约定**：
- 所有过程文档存放在 `{kb_dir}/` 目录下，该目录由调用方指定
- `{name}` 是需求/问题名称，用作目录名
- 模板中的路径需要替换为实际路径
